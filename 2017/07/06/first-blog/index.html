github+hexo搭建个人网站

2017.07.06 耿晓勇  热度 ℃

随着网络的不断发展，越来越多的人有了自己网站，不知你是否想要拥有一个属于自己的博客网站，如果有，我或许可以帮得到你，请跟随此文章快速搭建自己的博客网站，用文章来记录你的成长，搭建个人博客的方法有很多种，在这里，我使用的是github+hexo的方法进行搭建。

直接开始

注册Github并创建个人仓库

进入Github官网并进行注册，然后点击New repository创建仓库，仓库名为 用户名.github.io

Git安装

Git是一款分布式版本控制系统，免费、开源，可以敏捷高效的进行版本管理。了解更多Git的知识，各位可以自行查找。
首先从官网下载Git。
下载安装完成之后，在命令行中输入git --version测试是否安装成功，如出现下图所述，则安装成功。

之后鼠标右击或通过菜单搜索打开Git Bash，配置用户名和邮箱信息。

gti config --global user.name '用户名'

git config --global user.email '邮箱'

生成ssh秘钥：

ssh-kengen -t rsa -C 'github注册邮箱'

连续几个回车，生成的.ssh文件夹在系统盘用户根目录下：

在文件夹中找到id_rsa.pub，将内容复制

进入刚刚注册的Github中，settings-->SSH and GPG keys中点击New SSH key，title可填可不填，将刚才复制的内容添加到Key中，之后点击Add SSH key

回到Git Bash中输入：

ssh git@github.com

检测Github公钥是否添加成功，如下则说明添加成功。


以上就是关于Git的安装

Node.js安装

Node.js是运行在服务端的Javascript，官网下载地址（推荐安装msi格式）。接下来所使用Hexo就是基于Node.js。
下载安装完成之后，在命令行中分别输入node -v检测Node.js是否安装成功

Node.js中包含了环境变量以及npm的，在命令行中分别输入npm -v检测npm是否安装成功

到这一步，Node.js就安装完成了。

Hexo安装

在安装Hexo之前先在电脑中创建一个文件夹，命名可以是blog之类的，之后的Hexo框架和自己写的东西以及主题都会放在这个文件夹中，之后进入文件夹内，按住shift+鼠标右键，点击如下图：

输入命令安装hexo,由于国内网速原因，下载可能需要一定的时间，可以选择使用淘宝镜像进行下载。（之后的操作都在刚刚创建的文件夹目录下完成）

npm install -g hexo-cli

完成之后，输入hexo -v检查

安装相关工具：

npm install hexo --save

初始化博客：

hexo init

之后输入以下代码，出现下面图片中的内容之后，在浏览器输入localhost:4000，即可查看我们的网站雏形

hexo new test_my_site
hexo g
hexo s

下面这张图片就是我们网站的雏形

Ctrl+c即可退出预览。

网站推送

上面只是完成了在本地预览，接下的就需要将它推送的Github上，再次之前，需要说明的是：在我们的博客文件夹中有.config.yml文件，一个在文件夹根目录下，称之为站点配置文件

另一个在我们的主题文件夹themes内，称之为主题配置文件，后续修改主题内容都与它有关

之后打开站点配置文件，翻页到最后将其修改如下：
deploy:
type: git
repo: Github上仓库的完整地址，结尾处记得加上 .git
branch: master
注意：这里相对会严禁一些，分号后面必须加空格。
上述完成之后安装Git部署插件：

npm install hexo-deployer-git --save

初始化本地仓库：

git init

连接到远程仓库：

git remote add origin Github上仓库的完整地址，结尾处记得加上 **.git**

发布网站：

hexo clean && hexo g && hexo d

到这里，我们的网站就初步完成了，在浏览器输入用户名.github.io,就可以看到我们的网站了。为了让我们的网站更方便记忆接下来就是要进行域名绑定了，这里我推荐使用阿里云来购买域名，注册登录购买之后，进到域名列表，可以看到自己购买的域名，之后点击解析。

然后添加解析，如下：

添加解析记录，两处A的记录值是Github的地址，CNAME处的记录值是你的用户名.github.io,然后进入Github中，进入之前创建的仓库，点击settings，设置Custom domain，将自己注册好的域名填入，save

上述步骤完成之后，进入博客本地文件夹，在Blog/source目录下，新建记事本，将域名输入进去，保存为所有文件命名为CNAME，。之后执行以下步骤：

hexo clean && hexo g && hexo d

到这里，我们的博客就搭建完了。
